# Explosion (Minecraft Datapack)
If You Look at an Entity, it Explodes

## How to Use
1. Download and unzip the file.
2. Move the folder to `.minecraft\saves\{world name}\datapacks`.
3. Use `/reload` command in-game to apply the changes.
